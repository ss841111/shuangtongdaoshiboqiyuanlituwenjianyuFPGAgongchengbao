## 工程概述

- **核心芯片**: AD9288，一款高性能的模拟到数字转换器，广泛应用于高速数据采集系统。
- **开发环境**: 大部分工程基于Quartus，少量适用ISE或Vivado，确保兼容性与多样性。
- **资源内容**: 包括但不限于项目文档、原理图、 FPGA工程文件(VHDL或Verilog)。
